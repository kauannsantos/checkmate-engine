# checkmate-engine
Checkmate Engine é um projeto de xadrez em Java que une estratégia e inteligência artificial. Desenvolvido como forma de aprofundar meus conhecimentos em algoritmos e arquitetura de software, o motor utiliza Minimax com poda Alpha-Beta e heurísticas de avaliação para tomar decisões. Mais do que um jogo, é um laboratório de evolução técnica. ♟️
